# NIST COBOL Test Suite

This directory holds programs from the NIST COBOL 85 test suite.

Download from: https://www.itl.nist.gov/div897/ctg/cobol_form.htm

After downloading, extract the `.cob` / `.CBL` files here.
The suite covers all four COBOL 85 modules (nucleus, sequential I-O, relative I-O, indexed I-O)
and is the primary real-world corpus for parser and feature-extractor validation.
